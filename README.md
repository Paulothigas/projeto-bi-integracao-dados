#  Análise de Dados com Power BI e Python

##  Descrição
Este projeto acadêmico tem como foco a etapa de visualização e análise de dados de uma empresa, utilizando Power BI para geração de insights de negócio. A partir de dados tratados e integrados, foram desenvolvidos dashboards interativos com o objetivo de apoiar a tomada de decisão.

##  Objetivo
Analisar dados organizados e estruturados para identificar padrões, gerar indicadores e fornecer insights estratégicos relevantes para o contexto empresarial.

##  Tecnologias utilizadas
- Python (Pandas)
- Power BI
- Excel

##  Fontes de dados
- IBGE (dados populacionais)
- Polícia Federal (registros)

##  Etapas do projeto
- Coleta de dados de múltiplas fontes
- Tratamento e limpeza de dados (encoding, padronização de texto)
- Integração de bases utilizando código IBGE
- Estruturação dos dados para análise
- Desenvolvimento de dashboard interativo no Power BI

##  Principais análises
- Indicadores de negócio (CAC, LTV, conversão e NPS)
- Análise por localização geográfica
- Segmentação por perfil (gênero e idade)
- Identificação de padrões e comportamento


##  Estrutura do projeto
- `data_clean/`: dados tratados
- `notebook/`: tratamento e análise em Python
- `dashboards/`: arquivo Power BI

##  Resultado
O projeto demonstra a aplicação de técnicas de análise de dados e Business Intelligence para transformar dados em informações úteis, permitindo a geração de insights e apoio à tomada de decisão.
